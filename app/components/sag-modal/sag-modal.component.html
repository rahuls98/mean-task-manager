About

Square Away Gamification mode is a feature that helps users feel motivated to complete tasks. Once the feature is enabled, the process of task completion is gamified by the inclusion of a point scoring strategy. For every task completed, a score in the range [0,2] is assigned to the user, where the maximum and minimum scores achievable are 2 and 0 respectively.

How scoring is done

The score assignment for each completed task is done with respect to the due date for submission.
If a task is submitted

  • a day before the specified due date, score assigned = 2
  • on the specified due date, score assigned = 1
  • a day after the specified due date, score assigned = 0

It also depends on the categories of tasks the user wants to apply gamification to. Eg: Projects, Office Work, etc.

Activating gamification mode

Gamification mode is disabled for new users by default. Add a label to the list below and submit to activate gamification mode.
